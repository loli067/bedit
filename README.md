## bedit
A custom script for Android Boot image editor

## How to Install
```
sudo apt install git device-tree-compiler lz4 xz-utils zlib1g-dev openjdk-17-jdk gcc g++ python3 python-is-python3 p7zip-full android-sdk-libsparse-utils && git clone https://github.com/Logic067/bedit && cd bedit && sh setup.sh
```
## Usage 
```
bedit
```
## Note 
```Remember to type in .img in the choose img name option
Also to have everything in the current directory 
like the boot.img or recovery.img etc
and the unpacked_boot_image
```
